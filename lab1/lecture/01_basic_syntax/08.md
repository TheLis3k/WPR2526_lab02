Instrukcja `switch` jest rodzajem skondensowanej instrukcji warunkowej, którą zazwyczaj zastępujemy rozbudowane i wielokrotne  
użycia `else if`.


<?php
```php
$miasto = "hel";
```
switch ($miasto) {
  case 'warszawa': echo "Pochodzisz ze stolicy?"; break;
  case 'hel': echo "Mieszkasz nad morzem?"; break;
  case 'sanok': echo "A może w Bieszczadach?"; break;
  default: echo "Miasto nierozpoznane.";
}
?>
```
```

Poniższy kod:


<?php
```php
$a = 1;
```
if($a==1)
{
  echo("a jest równe 1");
}
if($a==3)
{
  echo("a jest równe 3");
}
if($a==11)
{
  echo("a jest równe 11");
}
?>
```
```

Można zastąpić przez:


<?php
```php
$a = 1;
```
switch($a) {
  case 1:
    echo("a jest równe 1");
    break;
  case 3:
    echo("a jest równe 3");
    break;
  case 11:
    echo("a jest równe 11");
    break;
}
?>
